Pro test - техническая подготовка QA

Веб-приложение Pro Test поможет начинающему и опытному QA-инженеру оценить уровень своих знаний. В данном приложении реализована регистрация и логинизация пользователя через google account, а также через e-mail и пароль. После чего пользователь может пройти тест и увидеть его результаты.

GoIT - курсы программирования

Многостраничный сайт с использованием HTML, CSS и JavaScript. На сайте есть форма обратной связи и pop-up, который появляется через 30 секунд вашего присутствия на сайте.

Также на сайте есть отдельная страница курсов, с последующим переходом на лендинг каждого отдельного курса. Постоянная рубрика Journal сделана в виде блога с полезной информацией для начинающих разработчиков.

На сайте есть отзывы выпускников, контакты для связи и события компании. Кнопка в правом углу переводит пользователя в телеграм-бот, где он может создать свой собственный сайт с автопроверкой заданий.
